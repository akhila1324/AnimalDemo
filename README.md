# AnimalDemo

A simple Java application that demonstrates the core object-oriented programming (OOP) concepts of **inheritance** and **runtime polymorphism**.

## Features

* **Inheritance:** Uses a base `Animal` class with derived subclasses `Dog` and `Cat`.
* **Polymorphism:** Each subclass overrides the `makeSound()` method.
* **Dynamic Method Binding:** An array of `Animal` objects is used to invoke the overridden methods dynamically at runtime.

## How to Run

1. Make sure you have the Java Development Kit (JDK) installed.
2. Compile the Java files:
   ```bash
   javac AnimalDemo.java
